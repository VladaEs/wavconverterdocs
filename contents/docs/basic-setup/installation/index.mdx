To build and run **MFC MP3/WAV Converter**, you need a Windows development environment with Visual Studio and the required C++ tools installed.

## Repository

GitHub Repository:

**https://github.com/VladaEs/MFCMP3WAVConverter**

## Prerequisites

<Step>
<StepItem title="Operating System">

* **Windows 10 / Windows 11**
* Visual Studio development environment

</StepItem>

<StepItem title="Visual Studio">

Install one of the following:

* **Visual Studio 2022** (Recommended)
* **Visual Studio 2019**

Required workload:

* **Desktop development with C++**
* **MFC and ATL support**

Verify your installation by creating a new MFC project.

</StepItem>

<StepItem title="Git">

```bash
git --version
```

Download Git from:

```text
https://git-scm.com/
```

</StepItem>

<StepItem title="Required Knowledge">

Basic understanding of:

* **C++**

* **Windows API**

</StepItem>
</Step>

## Installation

<Step>
<StepItem title="Clone the Repository">

Clone the project from GitHub:

```bash
git clone https://github.com/VladaEs/MFCMP3WAVConverter.git
```

Open the project directory:

```bash
cd MFCMP3WAVConverter
```

</StepItem>

<StepItem title="Open the Solution">

Open the Visual Studio solution file:

```text
diplom.sln
```

Visual Studio will automatically load all project dependencies and configuration files.

</StepItem>

<StepItem title="Build the Project">

Select your preferred configuration:

```text
Debug x86
```

or

```text
Release x86
```

Then build the solution:

```text
Build → Build Solution
```

or use:

```text
Ctrl + Shift + B
```

</StepItem>

<StepItem title="Run the Application">

Start the application from Visual Studio:

```text
Debug → Start Debugging
```

or press:

```text
F5
```

</StepItem>
</Step>

## Project Features

| Feature                | Description                                                      |
| ---------------------- | ---------------------------------------------------------------- |
| **MP3 Playback**       | Play MP3 audio files using Windows Media Player integration.     |
| **WAV Playback**       | Load and play WAV audio files.                                   |
| **Audio Conversion**   | Convert WAV files into the custom MUC format.                    |
| **MUC Decoding**       | Decode MUC files back into WAV audio.                            |
| **MDCT Transform**     | Modified Discrete Cosine Transform implementation.               |
| **Quantization**       | Frequency-domain quantization using scale factors.               |
| **Spectrum Analyzer**  | Real-time frequency visualization during playback.               |
| **ID3 Metadata**       | Read metadata such as title, artist, and album artwork.          |
| **Custom Audio Codec** | Experimental audio compression format developed for the project. |

## Development Notes

<Note title="Debug Configuration" type="warning">
The project is primarily developed and tested using the Debug x86 configuration. Some Release configurations may require additional optimization fixes.
</Note>

<Note title="Audio Processing" type="info">
The custom MUC codec uses MDCT, quantization, dequantization, windowing functions, and overlap-add reconstruction for audio compression and playback.
</Note>

## Building for Release

To create an optimized build:

```text
Build → Configuration Manager
```

Select:

```text
Release
```

and rebuild the solution.

## Troubleshooting

| Problem                    | Solution                                                    |
| -------------------------- | ----------------------------------------------------------- |
| Project won't build        | Verify MFC support is installed in Visual Studio.           |
| Missing headers            | Ensure all project files were cloned correctly.             |
| Audio doesn't play         | Check Windows Media Player components are available.        |
| Spectrum not visible       | Verify playback is active and MDCT data is being generated. |
| Conversion output is noisy | Check MDCT, quantization, and overlap-add implementation.   |
