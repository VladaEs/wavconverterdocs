## MUC Audio Codec

**MUC Audio Codec** is a custom audio compression project built in **C++**. It implements a complete audio processing pipeline including WAV parsing, frequency-domain compression, custom file formats, and audio playback.

Whether you're interested in digital signal processing, codec development, or custom media formats, MUC provides a practical implementation of the technologies used by modern audio codecs.

## Why use MUC?

This project was built to demonstrate the principles behind audio compression and digital signal processing. It provides a complete implementation of a custom codec, allowing developers and students to explore every stage of the encoding and decoding process.

Use it for:

- **Audio Compression Research** – Experiment with custom encoding algorithms
- **DSP Learning** – Understand MDCT, IMDCT, quantization, and windowing
- **Codec Development** – Explore the architecture of modern audio codecs
- **Educational Projects** – Learn how compressed audio formats work

## Core Features

| Feature | Description |
|----------|-------------|
| **WAV Support** | Read and write standard PCM WAV files. |
| **Custom MUC Format** | Store compressed audio using a proprietary container format. |
| **MDCT Compression** | Transform audio into the frequency domain for compression. |
| **IMDCT Reconstruction** | Recover PCM audio during decoding. |
| **Hann Window Processing** | Reduce artifacts between overlapping frames. |
| **Quantization** | Compress frequency coefficients efficiently. |
| **Music Playback** | Play WAV, MP3, and MUC files through a unified interface. |
| **Metadata Support** | Manage audio file information and custom headers. |

## Codec Pipeline

The MUC codec processes audio through several stages:

1. Load PCM audio samples from a WAV file.
2. Apply the Hann window function.
3. Perform MDCT transformation.
4. Quantize frequency coefficients.
5. Store compressed frames in a MUC container.
6. Reconstruct audio using IMDCT during decoding.

## Project Goals

The primary goal of MUC is educational. The project focuses on understanding audio compression techniques and implementing them from scratch in C++.

Topics covered include:

- WAV File Structure
- PCM Audio Processing
- MDCT and IMDCT
- Quantization
- Overlap-Add Reconstruction
- Custom File Formats
- Audio Playback Systems
