

The MFC MP3/WAV Converter project is designed around a modular audio processing pipeline. Most project configuration is located in the codec, audio, and application settings classes.

## Codec Settings

This section contains the primary audio codec configuration used during encoding and decoding.

```cpp showLineNumbers
static constexpr int windowSize = 2048;   // MDCT window size
static constexpr int Q = 127;             // Quantization coefficient
```

These values directly affect:

* Compression quality
* Frequency resolution
* File size
* Reconstruction accuracy

## Audio Configuration

The application currently supports PCM WAV processing and MP3 playback.

```cpp showLineNumbers
Sample Rate     : 44100 Hz
Channels        : Mono / Stereo
Bits Per Sample : 16-bit PCM
```

Supported formats:

| Format | Read | Play | Convert |
| ------ | ---- | ---- | ------- |
| WAV    | ✓    | ✓    | ✓       |
| MP3    | ✓    | ✓    | ✗       |
| MUC    | ✓    | ✓    | ✓       |

## MDCT Processing

The custom codec uses a Modified Discrete Cosine Transform (MDCT) to convert audio samples into the frequency domain.

```cpp showLineNumbers
Input Samples
    ↓
Hann Window
    ↓
MDCT
    ↓
Quantization
    ↓
MUC File
```

During decoding:

```cpp showLineNumbers
MUC File
    ↓
Dequantization
    ↓
IMDCT
    ↓
Overlap Add
    ↓
PCM Audio
```

## Window Function

The encoder applies a Hann window before each MDCT operation.

```cpp showLineNumbers
float w =
    0.5f *
    (1.0f - cos(
        2.0f * M_PI * i /
        (windowSize - 1)));
```

The window reduces spectral leakage and improves reconstruction quality.

## Quantization

Frequency coefficients are compressed using band-based quantization and scale factors.

```cpp showLineNumbers
output[i] =
    (int)(
        input[i] /
        maxVal *
        Q);
```

Each frequency band stores:

```cpp showLineNumbers
Scale Factor
Quantized Coefficients
```

This significantly reduces storage requirements.

## Spectrum Analyzer

The player includes a real-time frequency spectrum visualizer.

```cpp showLineNumbers
Current Position
    ↓
Audio Window
    ↓
MDCT
    ↓
Spectrum Data
    ↓
GDI Visualization
```

The visualizer updates during playback using timer-based refresh events.
